~ Frontend Development - blok 2, jaar 2 ~

De opdracht

Hier kregen we de opdracht om individueel een website te maken. Ik koos ervoor om mijn portfolio te maken. In de website moesten bepaalde elementen zitten zoals: transition, javascript, grid, media queries en een formulier.

Ik vond dit gelijk erg leuk om te doen omdat ik weer met code mocht werken en mijn eigen website mocht maken. Grid had ik snel onder de knie, maar media quries vind ik nu nog steeds erg lastig.
Daarnaast moest je je verdiepen in twee dingen, ik heb mij vooral verdiept in de transitions. Ik vond het erg leuk om hier mee te spelen, je kan er echt veel mee. Hieronder zie je een ding wat ik heb gedaan met de transitions.

Met Javascript wilde ik een hamburgermenu maken die alleen zichtbaar is als je op je telefoon zit. Ik had hier veel moeite mee omdat ik Javascript nog wel lastig vind. Uiteindelijk is het maken van een hamburger menu gelukt en als je erop klikt dan schuift het gehele menu uit. Het is mij alleen helaas niet gelukt in de tijd die ik had om het hamburgermenu alleen zichtbaar te maken bij het gebruik van een telefoon.

Voor de rest is de website ongeveer hetzelfde als deze. Ik ben opnieuw begonnen aan mijn portfolio omdat ik niet helemaal tevreden was en er te veel transitions in stonden.

Wat heb ik geleerd?

Ik heb erg veel geleerd bij dit vak. Ik ben beter geworden in coderen, ik kan steeds meer en hoef minder op te zoeken. Ook heb ik vooral geleerd hoe je dingen moet opzoeken op internet als je iets niet weet.
Daarnaast ben ik er bij dit vak achter gekomen dat ik verder wil gaan in frontend.
