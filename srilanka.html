~ Website Sri Lanka - blok 1, jaar 1 ~

De opdracht

We kregen de opdracht om individueel een website te maken met html en css. Je mocht zelf kiezen waar je het over ging doen. Ik heb gekozen om de website te maken over mijn reis door Sri Lanka omdat ik daar al veel content van had en hierdoor dus niet nog content hoefde op te zoeken.

Aan het begon vond ik dit erg moeilijk omdat ik nog nooit met html of css gewerkt had, ik had nog nooit gecodeerd. Toch vond ik het een hele uitdaging om mij hier meer in te verdiepen, ik vond het al snel interessant merkte ik.
Ik heb helaas geen afbeeldingen van het proces.

Bij de beoordeling kreeg ik feedback dat ik beter geen 'position absoluut' voor bepaalde dingen kon gebruiken omdat daardoor mijn footer niet goed uitkwam. Ik merkte dit zelf ook al bij het maken, maar wist niet goed hoe ik dit moest oplossen. Ik vroeg aan mijn docent hoe ik dit het best kon doen en ze zei tegen mij dat ik naar Flexbox moest kijken. Hier heb ik mij toen verder in verdiept heb ik het probleem zo opgelost.



Wat heb ik geleerd?

Dit was een erg uitdagende opdracht voor mij omdat ik erg weinig kennis hierover had. Ik heb vooral geleerd dat ondanks je weinig kennis hebt over iets je uiteindelijk toch in een korte tijd een goed resultaat kan neerzetten. Ook heb ik kennis gemaakt met programmeren. Het oneindig lang proberen van dingen en dat het uiteindelijk nog niet werkt omdat er een kleine fout zit in de code. Je code steeds opnieuw doorlezen. Maar vooral het trotse gevoel van iets hebben als het gelukt is en je weer een stap verder bent.
